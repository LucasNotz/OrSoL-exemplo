# Website da Organização do Sonhos Lúcidos

A OrSoL é uma iniciativa que visa disponibilizar aos interessados as informações necessárias para começar a explorar o mundo dos sonhos, além de contribuir para o desenvolvimento dessa área de pesquisa.

**Site em produção:** https://orsol.online

## 🚀 Funcionalidades

- HUB de informações sobre sonhos lúcidos
- Link para o instagram da OrSoL
- Participação em pesquisas e armazenamento em banco de dados (nuvem)
- Responsividade
- Integração com API (Astronomy Picture of the Day - NASA)

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- PHP
- Hospedagem na Hostinger
- Banco de Dados MySQL da Hostinger

Inteligência artificial foi utilizada em todas as etapas do projeto.
